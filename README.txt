
Программа-симулятор для Машины Различий Чарльза Беббиджа в рамках ролевой игры Стимпанк:
Лондонские хроники. Программа состоит из двух частей -- эмулятора машины и интерфейса
администрирования. Эмулятор -- полноценная вычислительная машина с консольным интерфейсом
и собственным языком программирования, в основе которого лежат запросы к базе данных. База
данных с информацией о жителях дополняется через специальный веб-интерфейс администратора.

В диретории documentation можно найти полную спецификацию языка программирования и тексты
про вычислительную науку на игре и экономику рассчетов на Машине Различий.

Программа требует интерпретатор Ruby и MySQL для работы.
Интерфейс администрирования -- простое веб-приложение, которое соединяется с той же базой данных.
